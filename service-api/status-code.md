#返回码文档
##一、服务返回码
###1.1 认证服务与用户服务返回码表
| code		|	描述
|
| 0010001	|	该用户已经存在
| 0010002	|	密码错误
| 0010003	|	密码为空
| 0010004	|	用户名与密码不匹配
| 0010005	|	用户不存在
| 0010006	|	用户名或密码为空
| 0010007	|	该邮箱不是默认邮箱，不能进行该操作
| 0010008	|	邮箱不存在
| 0010009	|	请用默认邮箱登录
| 0010010	|	邮箱没有通过验证
	



##二、系统状态码
| code		|	描述
|
| 0000000	|	操作成功
| 9999999	|	系统错误，请联系管理员
